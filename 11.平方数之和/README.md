给定一个非负整数 c ，你要判断是否存在两个整数 a 和 b，使得 a2 + b2 = c。
示例:
```c
输入: 5
输出: True
解释: 1 * 1 + 2 * 2 = 5

输入: 3
输出: False
```

***
主要解题思想：调用sqrt函数
